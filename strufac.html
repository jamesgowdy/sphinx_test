class strufac.ParsePDB(pdbfile)[source]¶

Parse a PDB file for info needed to calculate structure factors The PDB is parsed according to the following format described here The symmetry operators are extracted from the mandatory REMARK 290 record.

Parameters:

pdbfile (str) – path to a PDB file
Variables:
  • x – x-coordinate (cartesian world basis)
  • y – y-coordinate (cartesian world basis)
  • z – z-coordinate (cartesian world basis)
  • n – atomic occupancies
  • B – isotropic B-factors
  • e – element symbol
  • S – deorthoganlisation matrix
  • abc – unit cell lengths (in Angstroms)
  • angles – unit cell angles (in radians)
  • Rw – symmetry operator rotation matricies for cartesian basis
  • Tw – symmetry operator translation vector for catesian basis
Raises RuntimeError:
 

if no REMARK 290 SYMTRY record found
