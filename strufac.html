class strufac.ParsePDB(pdbfile)[source]¶
Parse a PDB file for data needed to calculate structure factors
The PDB is parsed according to the following format described at http://deposit.rcsb.org/adit/docs/pdb_atom_format.html

the symmetry operators are extracted from the mandatory REMARK 290 record.

Parameters:pdbfile – path to a PDB file :type pdbfile: str :ivar x: x-coordinate (cartesian world basis) :ivar y: y-coordinate (cartesian world basis) :ivar z: z-coordinate (cartesian world basis) :ivar n: atomic occupancies :ivar B: isotropic B-factors :ivar e: element symbol :ivar S: deorthoganlisation matrix :ivar abc: unit cell lengths (in Angstroms) :ivar angles: unit cell angles (in radians) :ivar Rw: symmetry operator rotation matricies for cartesian basis :ivar Tw: symmetry operator translation vector for catesian basis
