class strufac.ParsePDB(pdbfile)[source]¶

Parse a PDB file for info needed to calculate structure factors The PDB is parsed according to the following format described here. The space groups symmetry operators are extracted from the mandatory REMARK 290 record; the unit cell parameters are extracted from the CRYST1 record; the deorthoganilsation matrix is extracted from the SCALE record.

Parameters:

pdbfile (str) – path to a PDB file
Variables:
  • xw – x-coordinate (cartesian world basis)
  • yw – y-coordinate (cartesian world basis)
  • zw – z-coordinate (cartesian world basis)
  • n – atomic occupancies
  • Biso – isotropic B-factors
  • e – element symbol
  • S – deorthoganlisation (SCALE) matrix
  • abc – unit cell lengths (in Angstroms)
  • angles – unit cell angles (in radians)
  • Rw – symmetry operator rotation matricies for cartesian basis
  • tw – symmetry operator translation vector for catesian basis
Raises:
  • RuntimeError – if no REMARK 290 SYMTRY record found
  • RuntimeError – if no SCALE matrix found
  • RuntimeError – if no CRYST1 record found
  • AssertionError – if origin offset non-zero
